# Requirements

## v1 Requirements

### JLPT Content
- [x] CONT-01: User can browse JLPT levels N5 → N1
- [x] CONT-02: Each level has vocabulary, kanji, and grammar sections
- [ ] CONT-03: Vocabulary cards show word, reading, meaning, and example sentences
- [ ] CONT-04: Kanji detail shows on'yomi, kun'yomi, stroke count, meaning, and compounds
- [ ] CONT-05: Grammar points show pattern, explanation, and usage examples

### Quiz & Progression
- [ ] QUIZ-01: User can take quizzes per lesson
- [ ] QUIZ-02: Quizzes support multiple choice, fill-in-blank, and matching
- [ ] QUIZ-03: User sees instant feedback and score after each quiz
- [ ] QUIZ-04: Level advancement test unlocks next JLPT level

### User System
- [ ] USER-01: User can create account (email or Google)
- [ ] USER-02: User can log in / log out
- [ ] USER-03: User dashboard shows progress per level/lesson
- [ ] USER-04: User can bookmark/save vocabulary and grammar items

### Work Japanese
- [x] WORK-01: Dedicated business Japanese section
- [x] WORK-02: Keigo lessons (honorific, humble, polite forms)
- [x] WORK-03: Interview preparation Q&A
- [x] WORK-04: Workplace vocabulary and phrases
- [x] WORK-05: Resume/CV guidance

### Infrastructure
- [x] INFRA-01: Search across all content
- [x] INFRA-02: SEO metadata for all pages
- [x] INFRA-03: Responsive design (mobile + desktop)
- [x] INFRA-04: Deployable to Vercel (config prepared)

## v2 (Deferred)
- Audio pronunciations for vocabulary
- Spaced repetition review system
- Listening comprehension section
- Community features (comments, flashcards)
- JLPT mock timed tests

## Out of Scope
- Mobile native apps (PWA sufficient)
- Video lessons
- Tutor matching
- Downloadable content
